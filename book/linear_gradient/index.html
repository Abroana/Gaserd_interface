Problem:

linear-gradient in Android OS. example code:

.footer {
background: #e30a14;
background: linear-gradient(to right, #e30a14, #f39100);
}

But this don't work some devices with Android OS 4.0.3

Decision:

Fix it:

background: #e30a14;
background: -webkit-linear-gradient(left, #e30a14, #f39100);
background: linear-gradient(to right, #e30a14, #f39100);
